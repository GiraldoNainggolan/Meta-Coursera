Dalam kursus ini, Anda diperkenalkan dengan dasar-dasar pengembangan web. Mari kita lakukan rekap singkat tentang apa yang Anda bahas. Modul 1 dimulai dengan pengembangan web. Dalam modul pertama ini, Anda belajar tentang berbagai jenis pengembang web dan peran serta tanggung jawab pengembang front-end, back-end, dan full-stack. Anda memperoleh pengetahuan tentang teknologi inti HTML, CSS, dan JavaScript, dan mengeksplorasi konsep bagaimana Internet bekerja. Modul ini juga menampilkan contoh untuk menunjukkan seperti apa hari dalam kehidupan pengembang web. Selanjutnya, Anda belajar lebih banyak tentang halaman web, server web, dan browser web, dan bagaimana mereka semua bekerja sama untuk menciptakan apa yang kita sebut Internet. Anda membahas apa yang terjadi ketika Anda mengirim permintaan untuk mencari sesuatu secara online dan bagaimana informasi itu kembali kepada Anda. Kemudian Anda diperkenalkan dengan teknologi Internet inti dan bagaimana mereka berinteraksi. Anda menjelajahi beberapa alat browser web yang dapat Anda gunakan untuk membantu Anda membuat kode lebih efisien. Akhirnya, Anda belajar apa itu IDE dan bagaimana hal itu dapat membantu Anda mempercepat pekerjaan Anda untuk menulis kode. Pengantar HTML dan CSS diikuti Modul 2. Di Modul 2, Anda belajar tentang HTML dan CSS. Dalam HTML, Anda menemukan cara membuat dokumen HTML, menggunakan tag HTML sederhana dan dokumen tertaut untuk membuat situs web. Kemudian Anda belajar tentang Document Object Model atau DOM, dan bagaimana itu digunakan untuk menampilkan konten yang Anda lihat di perangkat Anda. Selain itu, Anda membahas teknik untuk meningkatkan aksesibilitas web dan beberapa teknologi bantu yang tersedia untuk menyediakan akses universal ke Internet. Selanjutnya, Anda belajar cara menggunakan CSS untuk menata halaman web. Anda menjelajahi cara memilih dan menata elemen dan menggunakan teks, warna, dan perataan untuk membuat halaman web yang Anda buat tampak hebat dan menawarkan pengalaman pengguna yang hebat. Anda kemudian menggunakan keterampilan HTML dan CSS baru Anda bersama-sama untuk membuat halaman web. Di Modul 3, Anda belajar cara menggunakan kerangka Bootstrap untuk membangun antarmuka responsif dan mengetahui manfaat bekerja dengan Kerangka Kerja UI. Pertama, Anda diperkenalkan dengan konsep desain responsif dan bekerja dengan perpustakaan. Kemudian Anda belajar bagaimana memulai dengan perpustakaan Bootstrap dan cara menggunakan gaya Bootstrap dan Anda menjelajahi beberapa kerangka kerja dan pustaka UI alternatif. Akhirnya, Anda diperkenalkan ke pustaka JavaScript react. Anda belajar mengapa itu dibuat oleh meta-engineer dan cara kerjanya pada tingkat tinggi. Anda kemudian secara singkat menjelajahi hierarki komponen DOM virtual dan beberapa alternatif untuk bereaksi. Dilakukan dengan baik dalam menyelesaikan rekap kursus ini. Sekarang saatnya untuk mempraktikkan semua yang telah Anda pelajari dalam penilaian bertingkat. Apakah Anda siap untuk mengedit halaman web biografi Anda? Semoga beruntung.
