Sekarang Anda telah belajar, bahwa Anda dapat membangun komponen di React untuk mengembangkan antarmuka pengguna. Tetapi bagaimana komponen React berubah menjadi elemen pada halaman web yang Anda gunakan. Ingat ketika browser web Anda menerima halaman HTML, itu membangun DOM, untuk mewakilinya. Tetapi memperbarui DOM dianggap mahal, karena sangat memakan waktu bagi browser web untuk melakukannya. Setiap kali browser DOM diperbarui, itu menyebabkan browser menghitung ulang halaman. Namun banyak situs web besar dan populer masih dimuat dalam waktu singkat hari ini. Bagaimana bisa? Dalam video ini, saya akan menjelaskan bagaimana React memecahkan masalah, dengan menghitung DOM virtualnya sendiri. Mari kita mulai dengan bagaimana komponen React berhubungan dengan situs web yang ditampilkan di browser Anda. Komponen realitas, memiliki hubungan satu lawan satu dengan elemen HTML yang ditampilkan di halaman web. Tetapi bagaimana React melacak elemen HTML mana yang perlu diperbarui. Di sinilah reaksi DOM virtual ikut bermain. Ketika React membangun pohon komponennya, ia membangun kubahnya sendiri di memori yang disebut DOM virtual. Virtual DOM adalah representasi dari browser DOM yang disimpan dalam memori. React menggunakan DOM virtual ini untuk memperbarui DOM browser, hanya jika perlu. Ini memastikan bahwa aplikasi Anda cepat dan responsif terhadap input pengguna. Biarkan saya menjelaskan cara kerjanya, React memeriksa untuk melihat apakah komponen HTML di DOM virtual cocok dengan DOM browser. Jika perubahan diperlukan, kubah browser diperbarui. Jika tidak ada yang berubah, maka tidak ada pembaruan yang dilakukan. Proses ini disebut rekonsiliasi. Mari kita uraikan apa yang terjadi ketika Anda memperbarui komponen dan React. Pertama, DOM virtual diperbarui. Kemudian React, membandingkan DOM virtual dengan versi sebelumnya dari DOM virtual, dan menentukan elemen mana yang telah berubah. Elemen berubah, dan hanya elemen-elemen yang diperbarui di browser DOM. Perubahan pada browser DOM, menyebabkan halaman web yang ditampilkan berubah. Dalam video ini, Anda telah belajar bahwa, React membangun dan memperbarui DOM virtual. Sehingga browser hanya akan memperbarui elemen HTML tertentu, alih-alih membuat ulang seluruh halaman.
