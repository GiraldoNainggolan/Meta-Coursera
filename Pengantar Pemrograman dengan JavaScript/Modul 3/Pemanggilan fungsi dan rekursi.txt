Treadmill adalah peralatan yang berguna untuk melakukan kardio di gym, atau saat berlari di luar ruangan bukanlah pilihan. Mereka juga mudah dimulai dengan menekan sebuah tombol. Namun bayangkan, jika tidak ada cara untuk mengontrol ketika treadmill berhenti, itu tidak akan ideal, bukan? Untungnya, sebagian besar treadmill memiliki fungsi yang memberitahu treadmill untuk berhenti setelah jangka waktu tertentu. Syukurlah. Dalam JavaScript, fungsi yang mengulangi tugas juga membantu, kecuali jika mereka berjalan tanpa henti. Dalam video ini, saya akan mendemonstrasikan apa itu fungsi rekursif, dan membantu Anda memahami cara menulisnya dengan benar untuk menghindari terjebak dalam loop tak terbatas. Memanggil, memanggil, atau menjalankan fungsi berarti menginstruksikannya untuk mengikuti kode di dalamnya satu baris pada satu waktu. Untuk mendemonstrasikannya, mari kita buat contoh terlebih dahulu. Ketik contoh fungsi, tanda kurung, lalu penyangga keriting kiri. Tekan enter untuk maju ke baris berikutnya dan ketik log titik konsol. Di dalam tanda kurung, ketik kutipan baris satu, hapus kutipan, lalu tutup dengan titik dua. Saya sekarang memiliki satu baris kode, tetapi mari tambahkan beberapa lagi. Pada baris terpisah, saya menempatkan dua baris log titik konsol lagi seperti yang saya lakukan sebelumnya, tetapi kali ini berisi string baris dua dan baris tiga masing-masing. Saya menutup fungsi dengan memasukkan penjepit keriting kanan. Saya telah membangun fungsi yang disebut contoh dengan tiga baris. Ketika saya menjalankannya, masing-masing baris akan dieksekusi satu per satu secara berurutan menghasilkan tiga string. Untuk membuat hal-hal menarik, mari tambahkan satu baris lagi ke fungsi contoh. Kali ini, ketik nama fungsi itu sendiri. Jadi itu contohnya, tanda kurung, titik koma. Sekarang, jika saya menjalankan fungsi lagi, itu akan berulang dalam loop tak terbatas. Jelas ini tidak akan berguna. Jadi mari kita tingkatkan, dan pastikan itu tidak akan berjalan tanpa henti. Saya menambahkan baris baru sebelum fungsi dan membiarkan counter sama tanda tiga semi titik dua. Selanjutnya, saya menghapus tiga baris log titik konsol saya, yang akan saya ganti dengan beberapa kode berbeda. Untuk baris pertama, saya mengetik log titik konsol, dan kemudian counter di dalam tanda kurung. Pada baris kedua, saya mengetik counter same sign, counter minus 1 semi colon. Dan akhirnya, untuk baris ketiga, saya mulai dengan mengetik if, dan kemudian di dalam tanda kurung, saya mengetik counter triple same sign, zero. Dan kemudian saya mengetik return di semi titik dua. Jika saya memanggil fungsi kali ini, itu akan mencatat angka 3, 2, 1, dan kemudian berhenti. Ketika sebuah fungsi memanggil dirinya sendiri, inilah yang dikenal sebagai rekursi. Rekursi adalah cara alternatif untuk menjalankan kode berulang tanpa menggunakan loop. Lain kali Anda melihat seseorang berlari di treadmill, bayangkan sebuah fungsi berjalan di latar belakang. Mungkin memanggil dirinya untuk terus berjalan sampai kondisinya dimaksudkan untuk berhenti. Dalam video ini, Anda belajar tentang penggunaan dan potensi masalah fungsi rekursif, dan cara menulisnya agar tidak berjalan tanpa henti.
