Pernahkah Anda memikirkan fungsi yang dilakukan remote TV Anda untuk Anda? Ini memungkinkan Anda untuk membuat perubahan pada TV Anda. Anda dapat menganggap DOM seperti remote TV yang memungkinkan Anda mengubah konten halaman web di layar. Bahkan memungkinkan Anda untuk mengganti hanya bagian-bagian tertentu dari halaman, seperti gambar atau judul atau keduanya. Namun, remote memiliki batasan tertentu. Anda hanya dapat mengubah saluran, kontras, kecerahan, dan volume. DOM memberi Anda kontrol yang jauh lebih halus daripada remote TV yang pernah bisa dilakukan. Misalnya, bayangkan Anda dapat mengubah seperti apa karakter dalam film TV dengan beberapa perintah sederhana saat film sedang diputar. DOM memungkinkan Anda untuk mengubah properti objek pada halaman web. Anda dapat menganggap DOM sebagai remote kekuatan super untuk situs web. Ini memberi pengembang kekuatan dalam bagaimana mereka dapat memanipulasi dan memperbarui halaman web. DOM adalah dalam bentuk objek JavaScript dengan objek ber sarang untuk berbagai bagian halaman. Objek ini memiliki objek bersarang sendiri sampai seluruh file HTML dipetakan dalam apa yang terlihat seperti struktur pohon. DOM adalah model file HTML yang disimpan sebagai objek JavaScript di memori browser Anda. Browser secara otomatis membangun DOM untuk setiap halaman web yang diunduh. Misalnya, jika Anda mengetik kan URL ke bilah alamat browser Anda, browser akan mengambil halaman web yang ada di domain ini. Jika browser terhubung ke server dan memungkinkan browser untuk mengunduh halaman, ia akan menerima semua kode HTML dan menyimpannya di memorinya. Browser kemudian akan menampilkan halaman yang diunduh. Ini juga akan membangun DOM halaman web itu sebagai model dokumen HTML yang baru saja diunduh browser Anda. Sebagai pengembang, Anda dapat berinteraksi dengan DOM halaman untuk membuat perubahan pada halaman web. Untuk berinteraksi dengan DOM, Anda dapat menggunakan tab Elements di dalam DevTools browser. Anda masuk ke DevTools dengan mengklik kanan di mana saja di jendela browser dan kemudian dengan mengklik “Periksa”. Tab Elemen memungkinkan Anda untuk berinteraksi dengan DOM dari halaman web yang saat ini aktif menggunakan antarmuka pengguna grafis, juga dikenal sebagai GUI. Browser juga memungkinkan Anda untuk berinteraksi dengan DOM menggunakan JavaScript. Untuk melakukan ini, Anda harus mengklik tab Konsol di DevTools browser. Anda dapat fokus pada panel Konsol dengan menekan tombol Escape pada keyboard Anda. Setelah selesai, Anda dapat mulai mengetik perintah JavaScript untuk melihat dan memanipulasi DOM. Ini mirip dengan bagaimana Anda berinteraksi dengan DOM menggunakan panel Elements, hanya saja kali ini Anda menggunakan kode untuk melakukannya.
