Salah satu fitur JavaScript yang paling kuat adalah manipulasi dom. Misalnya, Anda dapat mengklik tombol dan mengubah warna beberapa teks atau bahkan menampilkan pesan pop up. Pada dasarnya, Anda secara dinamis memperbarui konten html secara real time. Untuk melakukan ini, Anda harus dapat menemukan objek dalam dokumen html Anda yang ingin Anda manipulasi. Dalam video ini, saya akan memandu Anda melalui cara menggunakan pemilih dalam JavaScript untuk menemukan objek tertentu dengan cepat dalam dokumen. Selektor JavaScript bekerja dengan objek dokumen yang dapat Anda akses dengan mengetikkan dokumen kata kunci. Ini mengembalikan halaman web yang disimpan dalam memori browser yang dikenal sebagai model objek dokumen atau DOM. Atau, saya dapat menjalankan log dewan pada kata kunci dokumen untuk mendapatkan hasil yang sama. Mari kita mulai menemukan elemen tertentu di dalam objek dokumen menggunakan metode pemilih kueri. Misalnya, saya tahu halaman ini berisi dua elemen paragraf HTML, untuk mengakses yang pertama saya ketik document.queryselector. Kemudian dalam tanda kutip tunggal di dalam tanda kurung, saya mengetik huruf P. Ketika saya menekan enter untuk menjalankan perintah, ia mengembalikan elemen P pertama. Saya kemudian dapat memperluas ini untuk melihat teks paragraf yang terkandung di dalamnya, metode ini dapat digunakan dengan elemen html lainnya, seperti tag jangkar. Jika saya menjalankan perintah yang sama dengan huruf A, saya mendapatkan kembali elemen jangkar pertama di halaman. Ada pemilih JavaScript bernama sangat mirip yang memungkinkan saya mendapatkan semua keco cokan dari halaman web. Ini adalah metode pemilih kueri semua, untuk menunjukkan ini, saya mengetik pemilih kueri titik dokumen semua. Di dalam tanda kurung, saya meneruskan huruf p sebagai string dan memang saya mendapatkan kembali hasil yang menunjukkan bahwa ada dua tag paragraf di halaman ini. Pem@@ ilih JavaScript lain yang berguna adalah get element by ID yang dapat menemukan objek di fajar yang cocok dengan atribut ID html tertentu. Misalnya, katakanlah kita ingin menemukan sesuatu di halaman ini dengan ide judul, saya mengetik document.getelement dengan ID. Dan kemudian judul string dalam tanda kurung, saya menekan enter dan perhatikan bahwa elemen H1, dikembalikan ditetapkan dengan nilai atribut ID html dari heading. Selanjutnya, mari kita gunakan metode serupa yang mengembalikan elemen berdasarkan nama kelas tertentu daripada ID. Kita dapat melakukan ini dengan memanggil document.getElementsByClassName. Saya mengetik document.getElementsByClassName dan kemudian string txt dan tanda kurung. Sekali lagi, saya menekan enter dan perhatikan bahwa dua hasil dikembalikan. Dua tag paragraf yang berisi nama kelas txt. Jika Anda seorang pemula dalam perjalanan Anda dengan JavaScript, penting untuk dicatat bahwa elemen kata tunggal untuk ID dan jamak untuk nama kelas. Selain itu, misalkan elemen yang Anda cari tidak dapat ditemukan dalam hal ini Anda akan dikembalikan nol untuk ID. Koleksi kosong diwakili oleh tanda kurung persegi untuk nama kelas.
