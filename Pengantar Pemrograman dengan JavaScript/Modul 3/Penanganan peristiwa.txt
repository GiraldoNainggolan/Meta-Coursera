Katakanlah Anda menggunakan halaman web di komputer Anda dan Anda mengklik tombol, atau Anda mengetuk ikon suka pada gambar teman di ponsel Anda. Di JavaScript, klik tombol dan ketukan ikon seperti adalah contoh peristiwa yang dipicu pengguna. Peristiwa terjadi sepanjang waktu. Anda dapat menggunakan kode JavaScript untuk mendengarkan acara ini. Anda bahkan dapat memilih bagian halaman tempat Anda mendengarkan acara. Izinkan saya menjelaskan sebuah contoh. Anda mungkin ingin mendengarkan acara klik pada tombol Tambahkan ke Keranjang. Setelah Anda menangkap peristiwa seperti itu, Anda mungkin ingin menjalankan beberapa kode JavaScript. Misalnya, saya dapat menambahkan lingkaran dengan nomor satu di sebelah ikon keranjang belanja untuk menunjukkan bahwa sekarang ada satu item di keranjang belanja. Jika peristiwa yang sama dipicu atau dipicu lagi, kode penanganan acara kami kemudian menangani acara tersebut dengan memperbarui jumlah di lingkaran di sebelah ikon keranjang belanja. Lingkaran kemudian menampilkan angka dua untuk menunjukkan bahwa ada dua item di keranjang. Dalam JavaScript, fungsi yang menangani peristiwa yang ditangkap dikenal sebagai event handler. Izinkan saya mendemonstrasikan cara mendengarkan acara asing dengan menggunakan metode add event pendengar. Oke, jadi saya kembali ke konsol di sini dan izinkan saya mendemonstr asikan dua cara untuk mengatur pendengar acara untuk elemen HTML di halaman web. Salah satu cara termudah untuk mendengarkan acara adalah dengan menggunakan metode add event pendengar. Anda dapat melakukannya pada elemen HTML tertentu. Misalnya, misalkan saya ingin mendengarkan acara klik di badan situs web contoh ini. Pertama, saya perlu mendapatkan referensi ke elemen tubuh di mana kita ingin mendengarkan acara ini. Di konsol alat pengembang browser, saya mengetik document.queryselector, dan kemudian nama elemen di dalam tanda kurung. Untuk contoh ini, tubuhnya, dan saya lampirkan dengan tanda kutip tunggal. Selanjutnya, saya ingin menetapkan objek tubuh ke variabel. Saya mengetik const target sama untuk menetapkan objek tubuh ke variabel target. Saya telah menamai target variabel ini karena ini adalah target acara klik kami. Sekarang saya dapat membuat fungsi yang akan dijalankan ketika target ini diklik. Saya mengetik function handle click, dan kemudian di badan fungsi, saya mengkonsol mencatat string, mengklik badan. Langkah selanjutnya adalah menjalankan metode addEventListener pada elemen target. Saya mengetik target.add eventListener. Kemudian di dalam tanda kurung, saya memberikan dua argumen. Yang pertama adalah klik tipe acara sebagai nilai string, dan yang kedua adalah fungsi klik pegangan saya. Oke, jadi kode saya sekarang siap. Biarkan saya mengujinya dengan mengklik halaman web. Sukses. Perhatikan bahwa teks dikeluarkan ke konsol. Selain metode addEventListener, cara alternatif untuk mendengarkan acara adalah dengan menggunakan atribut acara HTML. Sebelum melakukan itu, izinkan saya terlebih dahulu menulis fungsi handler klik kedua bernama handleClick2. Sekali lagi, saya mengkonsol mencatat string di dalam badan fungsi. Kali ini akan menampilkan frasa yang diklik judul. Selanjutnya, saya perlu membuka panel elemen alat pengembang dan memperluas tampilan untuk elemen div, maka elemen heading harus terlihat. Saya klik kanan elemen H1 dan klik edit sebagai HTML. Kemudian setelah H1, saya membuat atribut dengan mengetikkan onclick sama diikuti oleh nama fungsi saya handleClick2 dengan sepasang tanda kurung di akhir. Akhirnya, saya klik di luar elemen untuk menyimpan perubahan dan kembali ke tab Konsol. Perhatikan bahwa ketika saya sekarang mengklik judul di halaman web, frasa mengklik judul dan mengklik badan dikeluarkan ke konsol. Pada dasarnya, klik tunggal ini telah memicu dua pendengar acara yang berbeda. Tetapi perhatikan bahwa jika saya mengklik di luar judul, itu hanya memicu pendengar acara yang mendengarkan klik pada seluruh elemen tubuh.
