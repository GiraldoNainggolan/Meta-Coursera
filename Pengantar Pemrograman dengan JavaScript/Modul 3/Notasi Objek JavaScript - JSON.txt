Dalam video ini akan membantu Anda belajar mengonversi string JSON ke objek JavaScript biasa dan sebaliknya. Jika Anda mengetikkan sepotong kode JSON ke konsol browser Anda, seharusnya tidak menghasilkan kesalahan apa pun. Misalnya, string JSON dengan satu properti bernama salam yang berisi nilai string hello. Jika saya menjalankan kode ini, saya mendapatkan kembali hasil yang sama. Perlu diingat bahwa JSON pada dasarnya hanyalah objek yang diwakili dalam bentuk string. Ini memiliki beberapa format khusus tetapi tetap berupa string. Jadi untuk bekerja dengan JSON, adalah umum untuk mengubahnya kembali ke objek JavaScript untuk bekerja dengan properti dan metodenya. Untuk melakukan ini, Anda perlu menggunakan objek JSON bawaan global dan metode penguraiannya, izinkan saya mendemonstrasikannya sekarang. Pertama saya akan menandatangani string JSON ke variabel bernama JsonStr menggunakan kata kunci const. Selanjutnya saya akan menjalankan metode parse JSON pada variabel ini dengan mengetik JSON.parse dan kemudian JSONStr di dalam tanda kurung. Saya menekan enter dan melihat bahwa objek JavaScript biasa dikembalikan. Saya ingin menyimpan objek ini sebagai variabel. Jadi saya menetapkan metode parse JSON ke variabel bernama aplainObj menggunakan kata kunci const. Oke, jadi sekarang string JSON saya disimpan dalam objek JavaScript biasa. Saya dapat memanipulasinya seperti objek JavaScript lainnya. Izinkan saya mendemonstrasikan ini sekarang dengan menetapkan kembali nilai properti salam, saya mengetik nama objek diikuti oleh titik dan kemudian properti objek sama dengan hi. Tekan enter dan perhatikan bahwa kata hi adalah output. Saya dapat mengkonfirmasi ini dengan memeriksa objek dan memperhatikan bahwa strukturnya telah diperbarui. Oke, begitulah cara Anda mengonversi string JSON menjadi objek JavaScript. Anda juga dapat pergi ke arah lain dan mengonversi objek biasa menjadi string JSON. Menggunakan string metode saya dari objek JSON. Biarkan saya menunjukkan ini sekarang. Sebagai contoh, mari kita mulai dengan mendeklarasikan objek bernama data dan menetapkannya beberapa properti dan nilai. Untuk menjalankan string JSON metode saya pada objek ini, saya akan mengetik JSON.stringfy dan menempatkan nama objek di dalam tanda kurung. Hasilnya adalah string JSON. Perhatikan bahwa kunci objek dan nilainya adalah string dikutip ganda dalam sintaks JSON. Penting untuk diingat bahwa sementara objek JavaScript biasa dapat menampung fungsi, string JSON tidak bisa. Batasan lain adalah JSON yang valid tidak mengizinkan penggunaan komentar JavaScript. Juga ketika Anda merangkai objek JavaScript yang berisi metode, metode itu akan dikecualikan dari operasi stringfy. Dalam video ini Anda telah belajar cara mengonversi string JSON ke objek JavaScript dengan metode parse. Dan bagaimana mengkonversi objek JavaScript ke string JSON menggunakan metode stringfy. Jika Anda bekerja dengan mengambil data dari API, mengubah string JSON menjadi objek JavaScript akan menjadi alur kerja standar. Anda kemudian dapat dengan mudah mengakses properti objek secara terprogram, ini adalah alat penting di sabuk alat Anda, dan saya mendorong Anda untuk mempraktikkannya.
