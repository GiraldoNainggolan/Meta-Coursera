Ketika dua pengembang JavaScript membahas subjek yang terkait dengan pekerjaan, terkadang Anda mungkin mendengar kata iterable disebutkan. Tapi apa sebenarnya arti iterable? Nah, dalam JavaScript iterable adalah tipe data apa pun yang dapat diiterasi menggunakan loop for of. Beberapa tipe data yang dapat Anda ulangi adalah array dan string. Dalam video ini, Anda akan belajar tentang karakteristik array string, dan Anda juga akan mengeksplorasi manipulasi string lebih lanjut menggunakan operator penggabungan. Dalam dunia JavaScript, sering dapat dikatakan bahwa string berperilaku seperti array karena string mirip array. Misalnya, Anda dapat menjalankan loop for di atas array huruf dan menampilkan hasilnya menggunakan metode console.log. Selain itu, Anda juga dapat menjalankan loop for di atas string. Misalnya, loop di atas string A, B, C untuk mengeluarkan huruf satu per satu. Izinkan saya mendemonstrasikan ini sekarang dengan beberapa contoh kode. Dalam file yang telah saya buka di konsol saya, saya memiliki variabel sayuran, yang memiliki array dengan tiga string. Setiap tali memegang nama sayuran, khususnya bawang, peterseli, dan wortel. Saya telah mengetik log konsol untuk veggies.length, yang akan mengembalikan jumlah item array. Saya juga memiliki log konsol untuk item pertama dan kedua dari array menggunakan indeks nol dan satu masing-masing. Akhirnya, saya memiliki contoh loop for yang menetapkan penghitung i menjadi nol. Kondisi keluar untuk loop ini adalah bahwa perbandingan i kurang dari panjang sayuran mengembalikan false dan i plus plus adalah nilai yang akan ditingkatkan pada setiap loop. Di dalam loop for, saya mengkonsol mencatat sayuran pada indeks i. Ketika saya menjalankan kode ini, saya mendapatkan tiga untuk mewakili panjang array. Bawang dari indeks 0, peterseli dari indeks 1 , dan akhirnya bawang, peterseli dan wortel sebagai hasil dari loop for yang mengeluarkan semua item dalam array. Sekarang, mari kita hapus konsol dan lanjutkan ke file berikutnya. Di sini saya akan mengeksplorasi bagaimana string dapat diulang. Dalam file ini, saya memiliki variabel bernama salam dengan nilai string howdy, bersama dengan log konsol untuk panjang string, dan karakter pada indeks nol dan satu. Sekali lagi, saya memiliki loop for yang secara struktural sama dengan contoh sebelumnya. Menjalankan kode ini memberi kita lima untuk panjang string H dan O untuk indeks nol dan satu dan kemudian masing-masing karakter string penuh secara ber urutan sebagai output dari loop. Meskipun perilaku serupa, string bukanlah array. Untuk memeriksa ini lebih dekat, mari kita beralih ke contoh berikutnya. Biarkan saya menghapus output terlebih dahulu. Dalam kode ini, saya memiliki variabel salam yang menyimpan string hello dan variabel pengguna yang menyimpan string Lisa. Kemudian saya menjalankan log konsol dengan string salam dan metode pop, yang seperti yang Anda tahu, menghapus item dari array. Ada dua baris kode lainnya, tetapi jangan khawatir tentang mereka sekarang, saya akan segera membahasnya.
