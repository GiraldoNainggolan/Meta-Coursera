Anda mungkin tahu bahwa kereta barang adalah sarana penting untuk memindahkan kargo. Tetapi pernahkah Anda berpikir tentang sistem yang membuat semuanya berjalan lancar saat perubahan dilakukan di setiap pemberhentian? Kereta biasanya diatur dalam urutan tertentu, dan setiap gerbong dapat membawa barang atau bahan yang berbeda. Gerbong lain juga dapat ditambahkan atau dil epas tanpa mengganggu sisa pesanan. Sama seperti kereta barang, JavaScript memiliki fungsi yang memungkinkan Anda untuk menyimpan dan mengatur ulang koleksi item yang diurutkan. Ini disebut array. Dalam video ini, saya akan memandu Anda melalui cara membuat array dan mengakses isinya. Saya juga akan menunjukkan bagaimana JavaScript menyimpan isi array dan mengindeks nilainya. Katakanlah Anda ingin meniru kereta barang mainan menggunakan JavaScript. Di kereta ini, setiap gerbong memiliki nomor yang dilukis di situsnya dan disajikan secara berurutan, mulai dari 0. Menggunakan JavaScript, Anda akan menetapkan beberapa kargo untuk setiap gerbong. Untuk yang pertama, mari kita masukkan var carriage0 sama dengan gandum. Hebat. Sekarang ketika Anda memeriksa isi kereta 0, Anda akan mendapatkan kembali kata gandum. Anda dapat menggunakan pendekatan yang sama untuk memodelkan seluruh kereta Anda. Katakanlah Anda melakukan hal itu dan berakhir dengan lima variabel dengan nilai yang berbeda. Anda baru saja membangun kereta dengan lima gerbong, masing-masing memegang kargo yang berbeda. Sekarang mari kita buat hal-hal menarik dan memperkenalkan kereta kedua. Yang ini hanya memiliki dua gerbong dengan angka 0 dan 1 dicat di sisinya. Tampaknya cukup sederhana, bukan? Nah, Anda tidak dapat menggunakan nama variabel carriage0 dan carriage1 karena sudah diambil. Anda dapat menggunakan nama yang lebih deskriptif, seperti train2carriage0 dan train2carriage1 tetapi rasanya agak berkata-kata. Masalah lain adalah Anda perlu memberi sinyal bahwa variabel- variabel ini adalah koleksi milik Train 2. Tetapi ketika ditulis dengan cara ini, JavaScript tidak mengerti itu. Untungnya, kita bisa memperbaikinya. Untuk membuat JavaScript memahami bahwa kita akan mengel ompokkan urutan variabel dalam koleksi, kita menggunakan array. Anda dapat membangun array menggunakan sintaks literal array, yang merupakan tanda kurung persegi pembuka diikuti oleh yang penutup. Array saat ini kosong, tetapi JavaScript sekarang memahami bahwa kami akan menyediakan koleksi item. Mari kita tetapkan koleksi itu ke variabel Train 1. Terakhir, mari tambahkan gerbong ke array kereta 1 dengan mengetikkan nama item dengan masing-masing dipisahkan oleh koma. Sekarang kita punya kereta api. Tapi apa yang terjadi dengan nomor gerbong? Tidak perlu khawatir. Di balik layar, JavaScript memberi setiap item angka mulai dari 0 yang sesuai dengan tempatnya dalam urutan. Anda kemudian dapat menggunakan nomor ini untuk mengakses setiap gerbong. Misalnya, mengetik train1 diikuti dengan 0, ter bungkus dalam tanda kurung akan mengakses nilai di dalam kereta 0, yang merupakan gandum dalam kasus ini. Untuk meringkas, array membantu kita mencapai beberapa hal dalam JavaScript. Array menandakan bahwa nilai yang ditetapkan semuanya milik grup. Nilai-nilai ini diurutkan dalam urutan dan masing-masing dapat diakses melalui nomor indeksnya dalam urutan itu. Dalam video ini, Anda belajar cara membangun array di JavaScript. Anda juga mempelajari bagaimana JavaScript mengurutkan item array menggunakan nilai indeks.
