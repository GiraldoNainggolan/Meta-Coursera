Berikut adalah solusi untuk latihan perulangan for dan while.

Tugas 1
12345
for (var i = 1; i <= 5; i++) {
    console.log(i);
};
console.log('Counting completed!');

Atur ulang
Tugas 2 
12345
for (var i = 5; i > 0; i--) {
    console.log(i);
};
console.log('Countdown finished!');

Atur ulang
Tugas 3
1234567
var i = 1;
while (i < 6) {
    console.log(i);
    i++;
};
console.log('Counting completed!');

Atur ulang
Catatan: Beri nama variabel pertambahan Anda i. Perbarui variabel tersebut dalam perulangan while menggunakan i++.

Tugas 4
Tulis perulangan "while" yang akan melakukan kode berulang yang sama persis seperti di bawah ini:

1234567
var i = 5;
while (i > 0) {
    console.log(i);
    i = i - 1;
};
console.log('Counting completed!');

Atur ulang
Catatan: Pada perulangan while, kurangi nilai i menggunakan: i = i - 1.
