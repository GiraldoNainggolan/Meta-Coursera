Anda siap untuk memulai pemrograman dengan JavaScript. Tetapi sebelum Anda dapat mulai mengembangkan proyek dengan JavaScript, Anda perlu mengatur perangkat lunak yang disebut VS atau Visual Studio Code. VS Code adalah editor kode yang digunakan pengembang untuk menulis kode JavaScript mereka. Pada akhir video ini, Anda akan dapat menginstal VS Code, Node.js, dan ekstensi Code Runner pada Sistem Operasi Windows Anda. Mulai file JavaScript baru di VS Code dan jalankan file menggunakan ekstensi Code Runner, dan jelaskan apa yang dilakukan console.log di JavaScript. Mari kita mulai dengan menjelajahi proses pengunduhan dan instalasi VS Code. Pertama, saya membuka mesin pencari di browser saya dan mencari istilah VS Code atau Visual Studio Code. Saya mengklik tautan pertama yang mengarahkan saya ke situs web resmi di code.visualstudio.com. Tautan ini mengarahkan saya ke beranda situs. Di beranda ini, saya memilih tombol Unduh untuk Windows. Sebuah pesan muncul di situs web VS Code yang menyatakan, terima kasih telah mengunduh VS Code untuk Windows. Saya kemudian dapat melacak kemajuan unduhan saya di browser saya. Setelah file diunduh, saya klik file untuk memulai proses instalasi. Perjanjian lisensi muncul. Saya membaca perjanjian dan memilih Berikutnya. Saya kemudian menerima tujuan instalasi default dan folder Start Menu. Saya juga memilih tugas tambahan berikut. Buat ikon desktop, tambahkan buka dengan tindakan kode ke menu konteks file Windows Explorer, tambahkan buka dengan tindakan kode ke menu konteks Direktori Windows Explorer. Setelah saya memilih tugas tambahan ini, saya klik “Next”. Sekarang saya telah mengonfigurasi tugas-tugas ini, saya memilih tombol Instal. Setelah diinstal, saya kemudian memeriksa peluncuran Visual Studio Code dan pilih Selesai. Program Visual Studio Code saya kemudian muncul di jendela baru; pada halaman Memulai. Dari sini, saya mengklik ikon paling bawah di sebelah kiri jendela Kode VS untuk membuka panel ekstensi. Setelah di panel, saya mengetik Code Runner untuk mencari ekstensi Code Runner. Saya kemudian memilih dan menginstal ekstensi ini dari hasil pencarian. Saat ekstensi diinstal, saya kembali ke browser saya dan mencari Node.js. Saya mengakses file dari situs resmi di Node js.org dan klik tautan unduhan langsung, pastikan untuk menginstal versi Windows. Sebuah pop-up muncul di layar untuk melacak kemajuan unduhan. Setelah file diunduh, saya memilih Buka file untuk membuka wizard pengaturan Node.js. Saya kemudian memilih Berikutnya dan menerima perjanjian lisensi. Saya kemudian mengklik “Instal” untuk memulai proses instalasi. Setelah proses instalasi selesai, saya klik tombol “Selesai” untuk menutup wizard pengaturan. Saya kemudian kembali ke VS Code, di mana proses instalasi Code Runner selesai. Untuk mengonfirmasi bahwa Code Runner diinstal, saya dapat memeriksa pesan berikut di layar yang menyatakan ekstensi ini diaktifkan secara global. Tepat di sebelah kanan tab Code Runner adalah garis untuk file. Saya klik dua kali ini untuk membuka file baru. Saya kemudian mengklik “Pilih Bahasa” dan memilih JavaScript dari daftar bahasa. Saya juga dapat mengetik JS di bilah pencarian untuk mempersempit hasil saya. Selanjutnya, saya menutup tab Code Runner dan klik ikon ekstensi untuk mematikan bilah sisi ekstensi. Saya kemudian mengklik tab “Lihat” dan memilih opsi Terminal untuk membuka terminal yang saya tutup sebelumnya. Aku harus membersihkan terminal. Saya dapat melakukan ini dengan mengetik clear dan menekan tombol Enter. Selanjutnya, saya mengarahkan kursor ke tab Output terminal. Saya kemudian mengklik, tahan, dan menyeretnya ke kanan untuk men ambatkannya di sisi kanan jendela Kode VS. Sekarang, saya menutup Terminal lagi. Output harus tetap berada di sisi kanan layar. Saya juga dapat menyesuaikan lebar output dengan menyeret split ke kiri atau kanan. Akhirnya, saya mengetikkan beberapa kode ke Tab Untitled-1 sebagai berikut, console.log, diikuti oleh sepasang tanda kurung di mana saya mengetik Hello World dalam tanda kutip tunggal. Untuk menjalankan kode ini, saya klik ikon “Play”. Saya juga dapat menggunakan tombol pintas, Control Alt N. Kata-kata Hello World muncul di output saya. Informasi lain juga muncul, tetapi Hello World adalah semua yang perlu saya fokuskan untuk saat ini. Itu saja. Saya telah berhasil mengatur VS Code dengan Code Runner. Anda sekarang tahu cara menginstal Visual Studio atau VS Code, Node.js, dan ekstensi Code Runner pada Sistem Operasi Windows Anda. Anda juga dapat memulai file JavaScript baru di VS Code dan menjalankan file menggunakan ekstensi Code Runner. Anda dapat menjelaskan apa yang dilakukan console.log di JavaScript. Kerja bagus.
