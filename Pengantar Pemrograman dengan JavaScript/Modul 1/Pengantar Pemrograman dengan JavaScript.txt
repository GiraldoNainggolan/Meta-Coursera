Begitu banyak dari apa yang Anda anggap remeh dalam kehidupan online sehari-hari Anda, seperti menggunakan mesin pencari, berbelanja, dan menggunakan media sosial, dimungkinkan oleh JavaScript. Hampir semua interaktivitas yang Anda alami secara online, seperti menu drop-down atau fungsi pelengkapan otomatis, difasilitasi oleh JavaScript. JavaScript paling terkenal sebagai bahasa pemrograman web. Tapi itu telah menjadi jauh lebih banyak. Sekarang, ini digunakan tidak hanya untuk pengembangan front-end, tetapi juga untuk pengembangan back-end, dan Anda bahkan dapat membuat aplikasi seluler menggunakan JavaScript dan React. Tanpa JavaScript, lanskap pengembangan perangkat lunak modern akan terlihat jauh berbeda. Karena ini adalah alat yang sangat kuat dan sangat integral dengan pengembangan perangkat lunak, penting bagi Anda, sebagai pengembang baru, untuk mengetahui cara kerjanya dan cara menggunakannya. Dalam kursus ini, dimulai dengan Modul 1, Anda akan membahas pengenalan JavaScript, yang akan membantu Anda memahami bagaimana itu digunakan di dunia nyata. Anda akan belajar cara mengatur Visual Studio Code dan menginstal kepunahan code runner. Modul ini juga memperkenalkan Anda untuk pemrograman dalam JavaScript, yang mencakup pembelajaran tentang variabel, tipe data, operator, string, boolean, dan angka. Setelah ini, Anda akan melanjutkan untuk belajar tentang pernyataan bersyarat dan loop. Dalam Modul 2, Anda akan membangun apa yang telah Anda pelajari di modul pertama, dengan mempelajari tentang array, objek, dan fungsi. Anda juga akan belajar tentang bug dan kesalahan. Dalam Modul 3, Anda akan belajar tentang pemrograman fungsional dan paradigma pemrograman berorientasi objek, yang mencakup pemanggilan fungsi, rekursi, dan ruang lingkup. Dengan pelingkupan, Anda akan belajar cara menggunakan var, let, dan const. Selanjutnya, Anda akan belajar tentang pemrograman berorientasi objek, dan Anda akan merancang dan membangun program berorientasi objek. Anda juga akan belajar tentang fitur JavaScript lanjutan, seperti merusak array dan objek menggunakan empat loop dan literal template. Selain itu, Anda akan belajar tentang struktur data dan operator spread dan rest. Anda juga akan belajar bagaimana JavaScript digunakan di browser. Dalam modul ini, Anda akan memiliki kesempatan untuk berlatih memperbarui konten halaman web. Modul 4 adalah semua tentang pengujian dan kompatibilitas. Oleh karena itu, Anda akan belajar tentang jenis pengujian dan Anda akan menulis tes unit. Anda juga akan diperkenalkan dengan Jest dan belajar cara menulis tes dengan Jest dalam sebuah latihan. Anda akan belajar tentang pengembangan yang digerakkan oleh tes dan mulai berlatih menerapkannya juga. Anda juga akan belajar tentang tantangan pengujian JavaScript, yang mencakup belajar tentang webpack, transpiling, dan bekerja dengan fungsi panah di React. Sebagai bagian dari modul ini, Anda juga akan belajar tentang lingkungan JavaScript lainnya, seperti Node dan NPM. Dalam Modul 5, Anda akan memiliki kesempatan untuk mem praktikkan keterampilan baru Anda dengan membuat pembuat tanda terima lemon kecil. Singkatnya, kursus ini memberi Anda pengenalan JavaScript. Ini adalah bagian dari program kursus yang menuntun Anda menuju karir dalam pengembangan perangkat lunak. Anda mungkin telah menemukan banyak kata teknis dan terminologi baru dalam video ini. Jangan khawatir jika Anda tidak sepenuhnya memahami semua istilah ini sekarang, semuanya akan segera menjadi jelas. Bagaimanapun, ini adalah bagian dari alasan mengapa Anda mengambil kursus ini.
