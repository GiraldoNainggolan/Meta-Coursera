Tipe data Boolean digunakan untuk memeriksa apakah pernyataan itu benar atau salah, yang menjadikannya bagian dasar dari mengetahui cara menggunakan JavaScript. Ini memiliki banyak kegunaan, tetapi dalam video ini, saya akan fokus pada bagaimana mendapatkan hasil perbandingan untuk menentukan apakah dua nilai sama atau tidak. Apa yang dimaksud dengan tipe data Boolean? Untuk memahami jawaban atas pertanyaan ini, mari kita pikirkan skenario kehidupan nyata membandingkan angka, seperti, misalnya, memeriksa apakah angka 1 lebih kecil dari angka 2. Dalam JavaScript, saya melakukannya seperti ini, 1 kurang dari tanda 2. Ketika saya menekan tombol Enter, JavaScript mengatakan bahwa ini adalah klaim yang benar. Mari kita lakukan beberapa perbandingan lainnya. Mari kita periksa apakah satu lebih besar dari dua, 1 lebih besar dari tanda 2. Kode ini mengembalikan nilai false dari tipe data Boolean. Tidak peduli bagaimana saya membandingkan angka, hasil perbandingan akan selalu benar atau salah. Tidak ada mungkin di sini. Itu sebabnya tipe data Boolean hanya memiliki dua nilai yang mungkin, benar dan salah. Selain operator kurang dari dan lebih besar dari, saya juga dapat menggunakan beberapa operator lain untuk membandingkan angka. Mungkin membantu untuk memikirkan berbagai operator perbandingan ini, tes tersebut. Katakanlah saya ingin menguji apakah perbandingan dua angka benar atau salah. Sekarang saya akan menjalankan beberapa tes dengan beberapa operator tambahan, dimulai dengan memeriksa kesetaraan, 1 sama dengan 2. Saya mengklaim bahwa 1 sama dengan 2 dan JavaScript mengembalikan false. Tidak, 1 tidak sama dengan 2. Klaim saya salah. Ketika pertama kali diperkenalkan dengan sintaks dengan dua tanda sama berturut-turut, tidak jarang orang sedikit bingung. Mengapa tanda ganda sama? Lagi pula, Anda tahu bahwa dalam matematika Anda hanya perlu satu tanda sama untuk menandakan kesetaraan. Tetapi dalam JavaScript, saya menggunakan tanda sama tunggal sebagai operator penugasan untuk menetapkan nilai dari sisi kanan operator penugasan ke variabel di sisi kiri operator penugasan. Misalnya, saya dapat mengetik skor var sama dengan 100 untuk mendeklarasikan variabel yang disebut skor dan menetapkannya ke nilai 100. Dengan kata lain, saya sekarang dapat mengonfirmasi bahwa nilai skor adalah 100 dengan mengetikkan skor di konsol. Memang 100 itu. Sebaliknya, dalam JavaScript operator kesetaraan harus memiliki tanda yang sama dan hanya memeriksa nilai. Selain itu, ada juga operator kesetaraan ketat yang memiliki tiga tanda sama dan memeriksa nilai dan jenisnya. Misalnya, jika saya menggunakan operator kesetaraan, izinkan saya mengetik 100 tanda sama ganda, kutipan ganda, angka 100, dan tutup kutipan ganda, konsol akan mengembalikan nilai Boolean dari true. Dengan menggunakan tanda ganda sama, saya hanya membandingkan nilai ini dan bukan jenisnya. JavaScript hanya memeriksa jika 100 sama dengan 100 terlepas dari tipe datanya. Ini mengabaikan fakta bahwa 100 di sebelah kiri operator kesetaraan tanpa tanda kutip ganda adalah angka, dan bahwa 100 di sebelah kanan operator kesetaraan dengan tanda kutip ganda adalah string. Operator kesetaraan yang ketat lebih tak kenal ampun. Ini memeriksa kualitas nilai dan kualitas jenis. Oleh karena itu, membandingkan angka 100 tanpa tanda kutip dengan string 100 dengan tanda kutip ganda akan mengembalikan nilai Boolean false. Mengapa ini salah? Itu salah karena meskipun nilainya sama, jenisnya tidak. Angka 100 tidak memiliki tipe yang sama dengan string 100. Ada juga operator yang kebalikan dari operator kesetaraan, operator ketidaksetaraan. Ini adalah kombinasi dari tanda seru dan tanda sama. Mari kita mengujinya. Saya mengambil nomor 1, tanda seru tanda sama dengan dan nomor 1 lagi. Ini mengembalikan nilai Boolean false karena salah untuk mengatakan bahwa satu tidak sama dengan satu. Selain itu, sama seperti ada operator kesetaraan ketat, ada juga kebalikannya dalam bentuk operator ketidaksetaraan ketat. Operator ini adalah tanda seru dengan dua tanda yang sama. Mari kita ketik nomor 1 sebagai angka, tanda seru tanda sama dua kali, dan kemudian nomor 1 di dalam tanda kutip ganda yaitu sebagai string. Ini mengembalikan nilai Boolean true karena nomor satu tidak sama dengan string 1 dengan nilai dan jenisnya. Memang benar bahwa kedua nilai ini tidak sama. Ada operator lain di JavaScript, dan ada beberapa konsep tambahan seperti prioritas operator dan asosiativitas operator, tetapi kita akan melewatkan ini untuk saat ini. Dengan contoh-contoh yang dibahas dalam video ini, Anda harus memiliki pemahaman dasar tentang cara kerja tipe data Boolean dan apa gunanya. Anda akan menemukan nilai Boolean lagi karena ada cara penting lain di mana Anda dapat menggunakannya dalam kode Anda selain membandingkan angka dan string.
