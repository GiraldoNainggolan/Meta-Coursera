Dalam video ini Anda akan belajar tentang jenis loop lain, loop while dan menggunakannya untuk mengkodekan hitungan mundur untuk ucapan selamat tahun baru. Loop while sangat mirip dengan loop for tetapi tidak persis sama. Perbedaan utama pertama adalah nilai counter. Dengan loop sementara ini diatur sebelum loop dan harus didefinisikan dengan jelas. Biarkan saya menunjukkan ini sekarang. Pertama, saya membuat variabel yang disebut counter dan mengatur nilainya ke 3. Selanjutnya saya ingin Anda tahu tentang perbedaan utama kedua. Ini adalah bahwa Anda hanya perlu menentukan kondisi keluar dalam tanda kurung deklarasi loop atau meletakkannya secara berbeda, kami hanya memberikan ekspresi JavaScript untuk diperiksa. Jika ekspresi itu mengembalikan true JavaScript kemudian akan menjalankan kode di dalam loop while. Dalam contoh ini kondisi kita akan menjadi counter lebih besar dari 0. Karena nilai penghitung 3 lebih besar dari 0 maka mengembalikan nilai true. Jadi untuk membangun loop while saya mengetik kata while dan kemudian kondisi di dalam tanda kurung, ini adalah counter lebih besar dari 0. Selanjutnya, di dalam blok kode loop liar saya perlu memberi tahu apa yang harus dilakukan. Dalam contoh ini saya ingin menjalankan metode console.log untuk menampilkan nilai penghitung, maka saya perlu mendekriminalisasi penghitung saya dengan nilai 1. Ini berarti bahwa pertama kali JavaScript mulai menjalankan baris kode dengan deklarasi loop while, ia memeriksa nilai variabel counter. Jika nilai penghitung lebih besar dari 0 dan memang demikian, itu akan menjalankan kode di dalam loop liar kemudian akan menampilkan nilai variabel counter saat ini ke konsol yang merupakan angka 3. Selanjutnya, ia akan menghitung nilai counter- 1, yaitu 3- 1 yang merupakan dua. Kemudian akan memperbarui variabel penghitung dengan menetapkan angka dua sebagai nilai barunya. Akhirnya akan kembali ke kondisi while dan memeriksa apakah nilai penghitung lebih besar dari 0, yaitu jika dua lebih besar dari 0. Karena itu, ia akan masuk ke blok kode while loops lagi dan mengeluarkan nilai saat ini penghitung yaitu dua. Kemudian akan memperbaruinya lagi dengan menetapkan hasil 2- 1 sebagai nilai yang diperbarui dari variabel counter. Sekarang karena sudah mencapai akhir blok, JavaScript akan kembali naik dan mengonfirmasi bahwa benar bahwa 1 lebih besar dari 0 kemudian masuk ke dalam body while loop lagi, menampilkan nilai 1 dan memperbarui penghitung menjadi 1- 1. Karena sudah mencapai akhir tubuh loop, ia akan naik kembali dan memeriksa apakah 0 lebih besar dari 0 dan karena tidak, itu tidak akan menjalankan kode di dalam loop while lagi. Sebaliknya kode apa pun di bawah penjepit keriting penutup dari loop while akan berjalan.
