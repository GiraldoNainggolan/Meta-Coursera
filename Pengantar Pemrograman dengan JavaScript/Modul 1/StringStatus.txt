Karena Anda baru mengenal JavaScript, Anda mungkin tidak terbiasa dengan string. String digunakan untuk mewakili dan bekerja dengan urutan karakter saat pemrograman dalam JavaScript. Biarkan saya menjelaskan bagaimana mereka bekerja. Untuk mendemonstrasikan konsep string, saya membuka alat pengembang di browser saya dan tab Konsol dipilih. Apa itu string? String dalam JavaScript adalah kumpulan karakter yang dilampirkan oleh tanda kutip tunggal, tanda kutip ganda. Kumpulan karakter semacam itu dikenal sebagai tipe data string. String adalah salah satu jenis data yang paling umum yang akan Anda kerjakan. Untuk mengkodekan string, saya mengetik sepasang tanda kutip tunggal. Sepotong kode ini dengan hanya kutipan tunggal pembukaan dan penutupan disebut sebagai string kosong. Untuk membuat string kosong, saya juga dapat menggunakan tanda kutip ganda. Sementara string kosong memiliki tempatnya di JavaScript, mereka tidak terlalu serbaguna. Tetapi string tidak harus kosong, jadi izinkan saya membuat string yang tidak kosong dengan beberapa huruf dan tanda seru. Saya melakukan ini dengan mengetik kata-kata halo di sana, dan tanda seru dalam sepasang tanda kutip tunggal. Selain huruf dan simbol tanda baca, kita dapat menambahkan hampir semua karakter lain ke dalam string, termasuk angka. Misalnya, saya bisa mengetik 'hello there! ' dan menindaklanjutinya dengan beberapa simbol dan angka. Ini masih string yang valid. Ada satu hal yang string tidak bisa lakukan dan itu adalah menerobos ke baris berikutnya. Ketika saya mengetik tanda kutip tunggal, kata halo, lalu tekan Enter sebelum mengetik tanda kutip kedua, itu menghasilkan kesalahan yang menghentikan kode berjalan. Hal yang sama terjadi jika saya menggunakan tanda kutip ganda. Ketika Anda menggunakan tanda kutip tunggal atau ganda untuk mengelilingi nilai string, karakter ini disebut sebagai pembatas. Ini karena saya menggunakannya untuk membatasi nilai string yang diberikan dari sisa kode saya. Ada beberapa situasi yang tidak biasa yang dapat membuat Anda bingung ketika Anda pertama kali mulai menggunakan string di JavaScript. Misalnya, katak anlah saya ingin membuat string baru. Saya melakukan ini dengan mengetikkan tanda kutip tunggal, diikuti oleh frasa, ini adalah hari yang indah, dan saya menutup string dengan tanda kutip tunggal kedua. Sekarang saya ingin menggunakan kontraksi itu dengan menggunakan apostrof tetapi sekali lagi, kode kesalahan. Ini karena JavaScript membaca kata itu, yaitu dua huruf pertama sebagai membentuk string, dan kemudian tidak memahami sisa kode karena apostrof. Untuk memperbaiki masalah ini, saya perlu memasukkan tanda kutip tunggal di dalam tanda kutip ganda. Saya melakukan ini dengan mengganti tanda kutip tunggal asli di kedua sisi frasa dengan tanda kutip ganda, dan sekarang Anda akan melihat tidak ada lagi kesalahan. Anda dapat membuat tanda kutip tunggal dan ganda juga tetapi lebih baik tetap menggunakan satu atau yang lain karena dapat membingungkan dan kurang konsistensi, dan begitulah. Anda telah mempelajari apa itu tipe data string dan apa itu string kosong. Anda juga dapat menjelaskan apa itu tipe data string di JavaScript, saya mendorong Anda untuk mencoba dan membuat beberapa string lagi sambil berlatih coding.
