Kapan pun Anda menggunakan browser, kapan pun Anda menjelajah Internet, Anda menggunakan JavaScript. Ini adalah bahasa internet. Nama saya Murtadha. Saya seorang insinyur perangkat lunak di Meta yang berbasis di kantor Seattle. Saya telah membuat kode dalam JavaScript sejak 2012, jadi JavaScript sebenarnya adalah sesuatu yang saat Anda menonton saya sekarang melalui video ini, itu adalah sesuatu yang mungkin Anda gunakan saat Anda membuat kopi pagi ini, mungkin pembuat kopi Anda memiliki beberapa JavaScript, mobil Anda mungkin memiliki JavaScript. Saat ini, kita berada di dunia di mana JavaScript sangat umum dan begitu digunakan di banyak teknologi dan banyak perangkat sehingga menjadi bahasa yang tidak dapat diganti. Pertama kali saya menggunakan JavaScript adalah, saya magang pada saat itu Facebook dan kemudian JavaScript tidak sepopuler dan umum. Saya tidak benar-benar mempelaj arinya di universitas atau di kursus saya. Bahasa lain yang lebih menarik atau menarik pada saat itu adalah orang yang menulis di Java atau C plus, plus. Orang-orang kadang-kadang tidak menganggap JavaScript sangat serius dan berpikir seperti itu adalah bahasa yang tidak digunakan oleh pengembang nyata dan serius. Tapi sekarang kita melihat dunia berjalan menggunakan JavaScript dan hampir setiap aplikasi yang kita berinteraksi dengan hari ini mungkin memiliki beberapa JavaScript di dalamnya, dengan satu atau lain cara. Apa hal yang membuat JavaScript begitu populer dan penting adalah bahwa itu adalah satu-satunya bahasa yang digunakan dalam pengembangan web. Dalam pengembangan game atau jenis aplikasi lainnya. Ada pilihan untuk menggunakan bahasa lain seperti Anda dapat menggunakan C plus plus atau Anda dapat menggunakan Java, atau Anda dapat menggunakan hal-hal lain. Tetapi ketika datang ke pengembangan web, satu-satunya bahasa yang tersedia adalah JavaScript, dan itulah yang membuatnya begitu populer dan umum dan penting untuk dipelajari ketika Anda mengembangkan aplikasi web. Di Metta, kami menggunakan JavaScript di seluruh tumpukan web kami, tetapi kami juga menggunakan JavaScript di tempat lain juga. Salah satu hal unik tentang JavaScript yang tidak dimiliki bahasa lain adalah bahwa itu adalah bahasa yang dapat digunakan untuk menulis front-end dan back-end, jadi banyak aplikasi, mereka memiliki back-end, mungkin dalam Python atau PHP atau Java atau yang lainnya. Tetapi mereka diharuskan memiliki front-end JavaScript karena itu adalah satu-satunya bahasa front-end. Tetapi karena JavaScript juga dapat digunakan pada backend jika Anda men desain aplikasi Anda dan jika Anda memilih untuk membuatnya secara konsisten JavaScript di seluruh tumpukan, kadang-kadang itu dapat menyederhanakan aplikasi dan kemudian Anda tidak perlu belajar banyak bahasa atau memelihara basis kode yang ditulis dalam bahasa yang berbeda. Beberapa tantangan yang saya hadapi ketika beralih di antara bahasa lain, seperti saya telah terbiasa dengan Java dan datang ke dunia JavaScript. Meskipun mereka tampak seperti mereka mungkin sangat mirip, mereka seperti setengah dari namanya sama. Tapi sebenarnya ternyata ada beberapa hal dalam JavaScript yang spesifik dan sangat unik untuk bahasanya. Ada hal-hal tertentu tentang JavaScript yang sedikit tidak konsisten jika dibandingkan dengan bahasa pemrograman lain seperti Anda memiliki C plus plus dan Java dan beberapa bahasa lain yang mengikuti perilaku tertentu, dan kemudian JavaScript terkadang berada di sudutnya sendiri melakukan hal-hal aneh. Tetapi saya sebenarnya menemukan bahwa sering kali JavaScript lebih mudah di pelajari daripada beberapa bahasa lain ini, jadi bahkan dalam keanehannya dan keunikannya dan beberapa hal yang membingungkan tentangnya, secara keseluruhan, saya menganggapnya sebagai bahasa yang lebih mudah di pelajari dan digunakan daripada yang lain ini. Saat Anda mempelajari JavaScript, Anda akan menemukan tantangan dan Anda mungkin menemukan bahwa segala sesuatunya tidak masuk akal. Saya mendorong Anda untuk menguasai mereka, dan Anda akan menemukan bahwa seiring waktu, Anda mungkin akan menikmati dan mencintai bahasa ini.
